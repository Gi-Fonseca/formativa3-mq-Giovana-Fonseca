# 🚀GalaxyTours
Este é um projeto desenvolvido para a avaliação formativa de Front-End, focado em deixar o site reponsivo utilizando Media Queries auxiliando uma agência de turismo espacial chamada de GalaxyTours

## 🛠️ Tecnologias Utilizadas
* HTML5 (Semântica e Acessibilidade)
* CSS3 (Estilização e Feedback Visual)
* JavaScript (Manipulação de DOM e Eventos)

## 📁 Estrutura do Projeto
O projeto foi organizado seguindo boas práticas de mercado:
/
├── index.html
└── src/
    ├── styles/  (Estilos da página)
    ├── scripts/ (Lógica de interação)
    └── img/  (Ativos visuais)

## 🚀 Funcionalidades Implementadas
* **Ponto de Quebra para Tablets (768px)**
Criando uma regra @media para telas com largura máxima de 768px, que quando a tela excede um limite ocorre a quebra de tela
* **Ajuste de Layout (Tablet)** Após a quebra, a tela reage com um ajuste de duas colunas
* **O Ponto de Quebra para Smartphones (480px) e Empilhamento Total (Mobile)** Criando uma regra @media para smartphones.  mude para 1 única coluna. Mudando a direção da estrutura flexível dele para que o título e a frase fiquem em uma coluna vertical, centralizando o texto.
* **impando a Tela (Mobile)**  Usando uma propriedade correta para esconder o elemento completamente na versão mobile.
